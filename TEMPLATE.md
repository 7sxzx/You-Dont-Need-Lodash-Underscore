### _.example
Explains or copy/paste lodash/underscore description.

  ```js
  // Lodash
  var result = _.example('abc', ...)
  console.log(result)
  // output: true

  // Native
  var result = 'abc'.example(...)
  console.log(result)
  // output: true
  ```
